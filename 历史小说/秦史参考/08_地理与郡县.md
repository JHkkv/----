## 三、关键地理节点

| 地名 | 重要性 |
|------|--------|
| **咸阳** | 京师。政治中心 |
| **骊山** | 嬴政陵墓所在。70万刑徒常年施工 |
| **沙丘** | 嬴政病逝处。沙丘之变发生地 |
| **大泽乡** | 陈胜吴广起义地 |
| **巨鹿** | 巨鹿之战→秦军主力被项羽击溃 |
| **函谷关** | 秦东面最重要的关隘 |
| **上郡** | 蒙恬驻军处。扶苏被贬监军地 |
| **沛县** | 刘邦故乡和起兵地 |
| **会稽/吴中** | 项羽起兵地 |

## 四、小说空间布局

**主角活动轨迹：**
- 统一前（1-100章）：主要在咸阳、关中、函谷关以东战场
- 统一后（101-400章）：咸阳朝堂→各地巡视→北疆（上郡）→东巡
- 崩塌期（401-500章）：游历全国，旁观帝国崩塌

**关键场景：**
- 咸阳宫→朝堂权谋主舞台
- 骊山→嬴政临终、魂魄入璧可能在此
- 沙丘→全书最关键的转折场景之一
- 大泽乡→"王侯将相宁有种乎"→秦末大乱的开端
