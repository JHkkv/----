# 秦朝文化与方士

> 此文件是轻玄幻线的历史基础

## 一、百家争鸣余波

秦统一时，诸子百家仍有余绪活跃：

| 学派 | 代表人物 | 秦廷态度 |
|------|----------|----------|
| **法家** | 商鞅（已故）、韩非（已故）、李斯 | **官方意识形态** |
| **儒家** | 淳于越、伏生 | 焚书令的重点打击对象 |
| **道家** | 黄老学派 | 与方士群体有交集 |
| **阴阳家** | 邹衍后学 | 五德终始说被秦廷采纳→秦为水德 |
| **墨家** | 分散流亡 | |
| **纵横家** | 衰落中 | |

### 博士官制度
秦廷设博士官70人，各学派均有名额——这是百家最后的官方保留地。焚书令未烧博士官所藏之书。

### 焚书令（前213）

实际内容：
- 焚烧《诗》《书》及百家语（非秦国史书以外的各国史书）
- 保留医药、卜筮、种树之书
- 博士官藏书不烧
- 令下三十日不烧者，黥为城旦

### 小说应用
- 汫述若在焚书令颁布时力谏→与嬴政/李斯正面冲突→政治危机
- 汫述暗中保护某些典籍/学者→隐藏任务
- 造纸术的出现→书籍复制成本降低→文化保存

## 二、方士与求仙——轻玄幻核心线

### 秦始皇的修仙行动

| 年份 | 事件 |

|------|------|------|

### 坑儒事件（前212）
- 起因：卢生、侯生逃亡。嬴政大怒→"诸生在咸阳者，吾使人廉问，或为妖言以乱黔首"
- 过程：御史审问→460余人被坑杀于咸阳
- 太子扶苏劝谏："诸生皆诵法孔子，今上皆重法绳之，臣恐天下不安"→嬴政怒，贬扶苏北监蒙恬军

### 小说应用——轻玄幻线

**秦朝的方士圈子并非全部是骗子。** 小说可设定：
- 少量方士确有真才实学——修真、炼丹、阵法的基础传承
- 秦始皇求长生是有真实依据的（只是方法不对或时机不对）
- 江述的系统和修为在方士群体眼中是某种"上古仙术"→引发各方士势力的觊觎
- 徐福的东海之行可以写成发现真实存在的"仙迹"（为后续剧情留伏笔）

**方士圈子的分化：**
- **求真派**：少数，求真正的道法传承
- **技术派**：有真才，如在炼丹/冶金/医术上有突破（江述可以合作）
- **骗术派**：纯粹骗吃骗喝（卢生/侯生之流）
- **政治派**：借方士身份影响朝堂

## 三、文字统一（书同文）

- 李斯作《仓颉篇》→小篆为标准字体
- 程邈创隶书→书写效率大幅提升（配合造纸术→行政效率飞跃）
- 江述贡献：造纸术+改良书写工具（如早期毛笔改良）→文教面板提升

## 四、秦宫廷文化

- **音乐**：秦已有较发达的音乐体系。高渐离击筑（为荆轲送行"风萧萧兮易水寒"）
- **建筑**：阿房宫规模宏大（"覆压三百余里，隔离天日"），骊山陵兵马俑世界奇迹
- **服饰**："衣服旌旗皆上黑"——秦为水德，尚黑

## 五、小说中的文化线

1. 江述的现代知识在秦人眼中=某种"异域秘术"→可融入方士群体的设定
2. 焚书令是江述与嬴政/李斯的正面冲突点之一→可写得很激烈
3. 方士圈子的分化→江述可以拉拢求真派、技术派→建立自己的情报网络
4. 造纸术→文教面板的根本性改变→长线伏笔
