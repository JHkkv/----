# 秦朝制度与律法

## 二、地方制度：郡县制

- 全国初设36郡，后增至40+郡
- 郡设**郡守**（行政）、**郡尉**（军事）、**监御史**（监察），三权分立
- 县设**县令**（万户以上）或**县长**（不足万户）
- 乡、亭、里为基层单位。亭长掌治安（刘邦曾任泗水亭长）

### 核心意义
**彻底废除了分封制**。自西周以来的"天子—诸侯—大夫—士"层级到此终结。皇帝直接任命郡守→权力直达基层。这是中国两千余年郡县制的起点。

### 小说应用
- 江述推行的改良政策如何通过郡县系统落实
- 地方郡守的贪腐vs中央的约束
- 统一后六国旧地推行秦法的阻力

## 三、军功爵制：二十等爵

### 核心规则

1. **斩一首，赐爵一级**——不论出身，奴隶也可获爵
2. **开放性**："宗室非有军功论，不得为属籍"——宗室无军功即削籍
3. **物质奖励**：爵一级→田一顷、宅九亩、仆一人
4. **世袭递减**：父爵死后传子需降级

|------|------|----------|
| 1 | 公士 | 田1顷，宅9亩 |
| 2 | 上造 | |

| 4 | 不更 | 可免徭役 |
| 5-9 | 大夫至五大夫 | 可食税邑 |
| 10-18 | 左庶长至大庶长 | 卿级军官 |
| 19 | 关内侯 | 食邑关内 |
| 20 | 彻侯 | 封侯，最高爵 |

### 小说应用
- 江述若出谋划策助秦军大胜→军功获取→系统奖励联动
- 六国旧贵族无法适应秦的军功爵制→矛盾冲突源

## 四、秦法核心

### 法家思想基础
- 源于商鞅，经韩非整合→"以法治国"，不别亲疏，不殊贵贱，一断于法
- "以刑去刑"：用重刑威慑→减少犯罪
- 轻罪重罚：偷一文钱可判黥面

### 常见刑罚

| 刑罚 | 适用 |
|------|------|
| 腰斩 | 重罪（李斯死于此刑） |
| 车裂（五马分尸） | 谋逆（商鞅、嫪毐死于此刑） |
| 黥面 | 较轻罪 |
| 刖刑 | 盗窃等 |
| 夷三族 | 大逆 |

### 连坐制
- **什伍连坐**：五家为保，十家相连。一家犯罪，其余九家若不告发则同罪
- **职务连坐**：上级对下属犯罪负有连带责任
- **族诛**：重大犯罪灭三族（父族、母族、妻族）

### 小说中的秦法
- 江述的系统改良方向：在秦法框架内减轻刑罚/增加柔韧性→与法家臣僚的冲突
- 扶苏主张宽仁→与秦始皇/李斯的矛盾→江述居中调和
- 百姓对秦法的恐惧vs对改良政策的期盼→民生线的情感基础
