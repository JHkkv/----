# 让AI帮你做简历，从扫描能力到输出成品全自动

找工作最痛苦的不是面试，是写简历。不知道自己会什么、经历怎么写、投出去没回音也不知道差在哪。

我做了这个Resume Forge Skill，6步全自动搞定：

---

📂 **第1步 能力扫描**
AI自动扫描你电脑上的项目文件，识别你真正会什么、用什么工具、做过什么——不是让你凭空想，是拿代码证据说话。

🔍 **第2步 简历考古**
自动搜索你写过的所有简历版本，提取可复用的经历描述，不用从头写。

🎯 **第3步 岗位匹配**
基于你的真实技能做三梯队推荐：高度匹配的、有竞争力的、可冲刺的。每个岗位附带薪资预期和简历关键词。

📝 **第4步 空白填充**
职业空白期太长、实习经历不够？搜索已注销的小企业作为经历参考背景。每条必须经你确认才写入。

🔧 **第5步 话术转化**
核心亮点——把你习以为常的表述变成HR想看的语言。比如"选品上架定价调价"→"全渠道内容矩阵策划与运营"。

✅ **第6步 质量审查**
7项自检：技能都有证据吗？数字前后一致吗？敏感信息没泄露吗？

---

输出MD（编辑）+ HTML（打印）+ TXT（平台粘贴）三种格式。

**怎么安装：**
① 复制这个Skill到 `.claude/skills/resume-forge/SKILL.md`
② 重启Claude Code
③ 输入 `/resume-forge 求职方向：xxx 目标城市：xxx`
