# 自媒体文案全流程创作助手

## 概述

一套完整的自媒体内容创作体系，分为两大模块：

| 模块 | 定位 | 解决什么问题 |
|------|------|-------------|
| **选题战略层** | 三层圈层矩阵选题法 | 选题枯竭、内容自嗨、更新断更 |
| **单篇执行层** | 调研→策划→撰写→排版配图 | 从零到一写完一篇笔记 |

**使用逻辑：** 当你需要"想方向"时启动战略层 → 确定选题后进入执行层完成单篇创作。

---

# 模块一：选题战略层（三层圈层矩阵选题法）

## 方法核心

一套**结构化要素拆解 + 跨层排列组合**的批量选题生产方法。将垂直赛道拆解为"核心词-属性要素-用户场景"三层矩阵，通过跨层关键词随机组合，快速批量生成既贴合赛道垂直性、又击中用户需求的选题。

**核心价值：** 将"依赖灵感的创意式选题"转化为"标准化的流水线拼接选题"。

## 方法架构

```
                    ┌─────────────────────┐
                    │   第三层：用户场景    │
                    │  ┌───────────────┐  │
                    │  │ 第二层：属性要素│  │
                    │  │ ┌───────────┐ │  │
                    │  │ │ 第一层     │ │  │
                    │  │ │ 赛道核心词 │ │  │
                    │  │ └───────────┘ │  │
                    │  └───────────────┘  │
                    └─────────────────────┘
```

- **核心层（圆心）**：赛道核心词，锚定垂直不跑偏
- **属性关联层（内圈）**：行业客观组成要素，提供内容骨架
- **用户场景层（外圈）**：用户视角需求维度，提供流量抓手

## 三步执行流程

### 第一步：搭建三层关键词矩阵

#### 第一层：核心层（圆心）

用 1-3 个词定义赛道核心主体。
示例：健身 → "健身/运动"、职场 → "职场/工作"、AI工具 → "AI/人工智能工具"

#### 第二层：属性关联层（内圈）

围绕核心行业，拆解直接相关的固有属性要素：

| 维度 | 说明 | 示例（健身赛道） |
|------|------|-----------------|
| 细分品类/分支 | 行业内的不同细分方向 | 减脂、增肌、瑜伽、普拉提、力量训练 |
| 核心技术/方法 | 该领域的核心操作 | 硬拉、深蹲、间歇训练、超级组、渐进超负荷 |
| 硬件/环境/工具 | 需要或相关的工具/场所 | 哑铃、弹力带、健身房、居家 |
| 效果/价值 | 行业能带来的结果 | 减重、塑形、体能提升、体态改善 |
| 标准/流程/规则 | 行业规范或通用步骤 | 饮食计划、训练周期、动作标准 |
| 时间/成本/价格 | 参数类要素 | 月卡费用、私教价格、每天投入时间、见效周期 |

每个维度拆解 3-8 个关键词，至少覆盖 4 个维度。

#### 第三层：用户场景层（外圈）

站在用户视角拆解需求维度：

| 维度 | 说明 | 示例（健身赛道） |
|------|------|-----------------|
| 人群分类 | 不同身份/基础的用户 | 新手小白、产后妈妈、上班族、学生党、大体重人群 |
| 场域场景 | 用户使用产品的场所 | 在家、公司、出差中、下班后、午休时间 |
| 动机场景 | 用户的目的和触发 | 拍婚纱照、体检亮了红灯、同学聚会 |
| 痛点场景 | 具体问题和障碍 | 坚持不下来、膝盖不好、怕练成肌肉腿、平台期 |

每个维度拆解 5-15 个关键词，至少覆盖 4 个维度。

### 第二步：跨层组合生成选题

从三层中各选 1 个关键词，通过以下句式串联成完整选题：

**7 种选题转化句式：**

| 句式 | 模板 | 适用场景 |
|------|------|---------|
| **提问式** | [人群] + [属性] + 怎么办/怎么选/多久见效？ | 教程/科普类 |
| **避坑式** | [人群] + [属性] + 最大的坑/千万别/99%的人都错了 | 带警示感，点击率高 |
| **对比式** | [场景A]和[场景B] + [属性] + 哪个更好？ | 制造冲突和选择感 |
| **揭秘式** | 为什么[人群/场景] + 一定要/绝不能 + [属性]？ | 反常识，有信息差 |
| **清单式** | [人群] + 必知的 + [N]个[属性] + 方法/技巧 | 收藏率高，适合涨粉 |
| **故事式** | [人群] + [属性] + 经历了[时间] + 发生了什么？ | 共鸣强，适合人设型 |
| **挑战式** | [时间/数量] + [属性] + 挑战 + 能变成什么样？ | 参与感强，适合系列 |

**生成规则：**
- 每轮生成 **15-25 个选题**，覆盖不同句式
- 同一选题避免两个同层关键词
- 选题口语化，像用户会搜或会问的问题
- 标注每个选题的句式类型和组合公式

### 第三步：热点叠加与优先级排序

结合当前热点对选题排序：

| 优先级 | 标签 | 说明 | 发布窗口 |
|--------|------|------|---------|
| P0 | 🔥 热点直击 | 选题直接关联当前热点 | 3-5天内 |
| P1 | 🔄 热点嫁接 | 可用热点角度切入 | 1周内 |
| P2 | ⭐ 爆款常青 | 高搜索量/强痛点/强共鸣 | 随时可发 |
| P3 | 📅 系列规划 | 可做连载的内容方向 | 排入日历 |

### 选题质量标准

每个选题须满足至少 3 项：
- [ ] **垂直性** — 锚定核心层，不跑偏赛道
- [ ] **用户相关性** — 绑定至少一个用户场景层关键词
- [ ] **可搜索性** — 包含用户会主动搜索的词
- [ ] **好奇心驱动** — 让人想知道答案（提问/揭秘/对比句式天然带钩子）
- [ ] **可执行性** — 实际能做出来，不是空想
- [ ] **差异化** — 与同赛道常见内容有区别

## 跨赛道复用

| 层级 | 复用程度 | 操作 |
|------|---------|------|
| 第一层（核心层） | 0% — 完全替换 | 换成新赛道核心词 |
| 第二层（属性层） | 20-30% — 框架复用 | 同维度逻辑拆解新赛道关键词 |
| 第三层（用户场景层） | 60-80% — 高度复用 | 人群/场景/动机/痛点维度可通用 |

# 模块二：单篇执行层（文案创作工作流）

## 适用场景
- 已有明确选题，需要从调研到排版的全流程创作
- 30天系列内容的规划与输出
- 自媒体新人的内容定位与风格探索

## 执行流程

```
调研 → 内容策划 → 文案撰写 → 排版与配图
```

### 第一步：调研
- 搜索小红书当前关于这个话题的热门内容和标签
- 分析爆款标题的共同特点
- 确认提及的工具和数据都是真实存在的

### 第二步：内容策划
- 提供 **3 个切入角度**，推荐最优并说明理由
- 设计内容大纲（分段结构 + 每段要点）
- 给出 **3-5 个标题备选**（每个 ≤ 20 字）

### 第三步：文案撰写
- 按小红书调性写正文（口语化、段落短、有情绪）
- **杜绝 AI 塑料感表达**（禁用"家人们""绝绝子""在当今"等）
- **70% 干货 + 30% 情绪**
- 文末附互动引导 + 3-5 个热门标签

### 第四步：排版与配图
- 给出分段排版策略（每段行数、emoji 位置、加粗策略）
- 设计 **5-6 张配图方案**（含封面），每张具体到可执行级别
- 推荐配色方案和制作工具

## 通用 Prompt 模板

将以下 Prompt 复制到任意 AI 助手（DeepSeek、豆包、Kimi、ChatGPT、Claude等），按需替换方括号中的内容：

```
你是一位专业的小红书内容创作助手。请按以下流程帮我完成一篇小红书笔记：

【我的需求】
主题：[填写你的内容主题]
系列背景（如有）：[如"30天AI教我做自媒体的第X天"]
目标读者：[如"自媒体零基础小白"]
我的风格偏好：[如"真实、接地气、不贩卖焦虑"]

【请按以下步骤执行】

第一步：调研
- 搜索小红书当前关于这个话题的热门内容和标签
- 分析爆款标题的共同特点
- 确认提及的工具和数据都是真实存在的

第二步：内容策划
- 提供3个切入角度，推荐最优并说明理由
- 设计内容大纲（分段结构+每段要点）
- 给出3-5个标题备选（每个≤20字）

第三步：文案撰写
- 按小红书调性写正文（口语化、段落短、有情绪）
- 杜绝AI塑料感表达（禁用"家人们""绝绝子""在当今"等）
- 70%干货 + 30%情绪
- 文末附互动引导 + 3-5个热门标签

第四步：排版与配图

- 设计5-6张配图方案（含封面），每张图具体到可执行级别
- 推荐配色方案和制作工具

【内容质量标准】
- 标题≤20字，带emoji
- 正文段落适配手机阅读（3-5行一段）
- 所有工具和数据真实可查，不编造
- 诚实定调，不神化AI能力
```

---

## 三条核心原则

1. **真实大于完美** — 工具和数据必须真实存在，不编造功能
2. **诚实定调** — 不夸大 AI 能力，建议"不一定对但当下最适合你"
3. **活人感优先** — 配图优先实拍，文字像真人说话，拒绝 AI 模板味

## 选题 × 文案衔接建议

- 选题战略层产出的每个选题，都可以直接代入模块二的 Prompt 模板的 `[主题]` 字段
- 连续日更时，用模块一规划 7-15 天的选题日历 → 每天用模块二快速执行
- 发现某个选题数据特别好 → 回到模块一的矩阵，围绕该选题的关键词纵向深挖同类选题

## 注意事项
- 所有建议仅供参考，最终内容由你自己决策
- 如需系列内容规划，建议提前明确每期的叙事主线
- 跨行业使用：只需替换模块一的第一层+第二层关键词，第三层可复用
