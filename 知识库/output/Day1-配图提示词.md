## 图 3：AI 对话截图

**手机截图 + 醒图后期：**

```
操作步骤：

1. 打开你和 AI（Claude/ChatGPT）的对话记录
2. 滚动到"五个问题"那一段
3. 截长图（或选取问题+你的回答中最有代表性的 3-5 轮对话）
4. 导入醒图

醒图处理参数：
- 滤镜：原生 → 调整 → "轻胶片" 30%
- 个人头像/昵称 → 用「涂鸦笔」涂抹遮盖（保护隐私）
- 在截图上叠加半透明矩形框，高亮最想展示的对话段落：
  矩形：颜色 #E8A87C，透明度 15%，圆角 12px
- 底部叠加一条白色渐变条（从透明到 #FFF8F0），让截图自然收尾
- 不额外加字（截图内容本身就是信息）
```

## 图 4：项目/作品截图

**手机或电脑截图 + 醒图后期：**

```
操作步骤：

1. 截取你用 vibe coding 做过的项目页面（网页 / 工具界面）
2. 如果没有线上作品，截取：
   - Claude/Cursor 的对话界面（展示你跟 AI 协作的过程）
   - 或你做的任何一个可视化的 AI 产出

醒图处理参数：
- 裁剪为 3:4 比例，保留核心界面
- 滤镜：原生 → 调整 → "清晰" 40%
- 在截图上方叠加文字标签：

  字体：黄油溏心体
  字号：26px
  颜色：#2C2C2C
  背景：白色圆角矩形 #FFFFFF，透明度 80%，圆角 8px，padding 12px
  内容：我的第一个 vibe coding 项目 ←

- 标签放在界面的"留白区域"上，不要遮挡核心内容
- 如果截图为暗色模式（黑底），则将标签改为深色底白字
```

## 图 5：环境/氛围照

**手机实拍 + 醒图调色：**

```
拍摄内容（三选一，选你最容易拍的）：

A. 桌面俯拍：
   - 一台亮着屏幕的电脑（屏幕显示 AI 对话界面）
   - 旁边放一杯水/咖啡
   - 一本摊开的笔记本 + 一支笔
   - 自然光从侧面打进来

B. 窗边侧拍：
   - 你坐在桌前，窗外有光
   - 不露脸（拍到手部+桌面即可）
   - 电脑屏幕亮着，上面是 AI 对话

C. 手持手机对镜子拍：
   - 镜子里的你拿着手机
   - 手机屏幕上显示小红书发布界面
   - 代表"从这一刻开始记录"

醒图调色参数：
- 滤镜：胶片 → "Portra 160" 40%
- 光感：+15
- 对比度：-10
- 色温：+8（偏暖）
- 颗粒：+15（增加日记质感）
- 暗角：+10

- 不额外加字，保留画面留白
- 出口：1080 × 1440px
```

## 图 6：收尾仪式感

**手机备忘录截图 / 手写拍照：**

```
方案A — 手机备忘录截图：

1. 打开手机自带备忘录
2. 背景色设为米白色（iOS 备忘录有"浅色"选项）

Day 1 ✅

找到了一个之前没留意到的东西——
我其实不是不会，是还没开始。

5. 醒图叠加一层淡淡的暖色：
   滤镜 → 轻胶片 20%

方案B — 手写拍照（推荐）：

1. 拿一张空白纸/便签
2. 用黑笔手写：

Day 1 ✓
和AI聊了五个问题，发现自己在指挥AI的时候最不迷茫。
明天试一件没做过的事。

3. 放在木桌/书本上，自然光拍摄
4. 醒图调色同上（图 5 参数），颗粒感可以更重一点（+20）

## 整体检查清单

在发布前核对：
- [ ] 6 张图色调统一（暖米白底 + 深灰字 + 暖杏点缀）
- [ ] 所有文字可读（字号不低于 24px，对比度足够）
- [ ] 隐私信息已遮盖（截图中的头像/昵称/手机号）
- [ ] 封面图能在一屏内看完（大字突出，小字不挤）
- [ ] 每张图都有相同的"日记感"——不是精致广告风，是真实生活感
- [ ] 正文+配图加起来 7-8 秒能刷完（不宜超过 9 张）
