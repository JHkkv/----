# Coze 学习路径

## 六个阶段

| 阶段 | 内容 | 关键技能 |
|------|------|----------|
| 一、入门 | 认识界面、搭建首个智能体 | 智能体 vs 工作流、变量 |
| 二、数据 | 字符串、数字、数组、对象 | 三个实操项目练手 |
| 三、逻辑 | 判断、循环、数据库 | 工作流具备思考+记忆 |
| 四、大模型 | token、参数调优、提示词 | 工具调用、知识库搭建 |
| 五、商业 | HTTP 调用、多模态生成 | 图像、视频、播客自动化 |
| 六、拓展 | 飞书联动、AI 辅助编程 | 企业级自动化、零代码编程 |

## 学习建议
- 每天 1-2 小时，预计 2 个月完成
- 方法：先动手做 → 再理解原理 → 然后自己复现一遍 → 最后用到实际场景
- 避免碎片化——跟着一条路径走到底，不要东学一点西学一点
