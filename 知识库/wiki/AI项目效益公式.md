# AI项目效益第一性公式

## 定义

从第一性原理推导出的AI项目效益评估公式，由 Hermes Agent 在10Agent头脑风暴中提出。

## 核心公式

```
AI效益 = (Q × S × C) − (D + O + M)
```

| 因子 | 全称 | 含义 |
|------|------|------|
| **Q** | Quality 质量提升 | AI产出相比人工/旧系统的质量倍数 |
| **S** | Speed 速度提升 | AI产出相比人工/旧系统的速度倍数 |
| **C** | Coverage 覆盖规模 | AI可处理的场景/用户量级 |
| **D** | Development 建设成本 | 数据获取+模型训练+开发+集成的一次性投入 |
| **O** | Operation 运营成本 | 推理+监控+更新+人工审核的持续性支出 |
| **M** | Maturity 组织成熟度 | 培训+变革管理+信任建设的**摩擦成本** |

## 关键洞察

- **分子 (Q×S×C)**：大多数项目**高估**了这部分
- **分母 (D+O+M)**：M（组织摩擦）被**严重低估**，这是80%项目ROI不及预期的根因
- 如果 AI方案的全生命周期成本 > 人工成本，项目就不成立（除非开辟全新收入）

## 与Claude方公式的融合

```
融合版 = [(Q × S × C) × 实现概率] − [显性成本 × 隐藏成本系数(2-4x) + M]

其中隐藏成本系数 = 需求返工(1.3-1.5x) + 集成复杂度(1.2-1.4x) + 人员流动(1.1-1.3x) + 规模化非线性(1.2-1.5x) + 合规补课(1.1-1.3x)
```
