# 智投 — AI求职招呼语生成器

> 上传简历，粘贴JD，一键生成高回复率的BOSS直聘招呼语。
> 
> 🔗 **在线使用**: [https://jhkkv.github.io/----/](https://jhkkv.github.io/----/)

## ✨ 功能

- 📄 **简历分析**: 上传 PDF / Word / TXT 简历，AI自动解析并填充你的背景信息
- ✍️ **招呼语生成**: 基于6种钩子策略，生成3个不同风格的个性化招呼语
- 📋 **一键复制**: 复制生成的内容到剪贴板，直接粘贴到BOSS直聘
- 📱 **移动端可用**: PWA设计，手机浏览器打开即可用，支持添加到主屏幕
- 🔒 **隐私优先**: 所有数据存储在浏览器本地，不上传任何服务器（仅发送脱敏文本到AI API）
- 🌐 **零依赖**: 单文件HTML，无需安装任何软件

## 🚀 快速开始

### 方式一：在线使用（推荐）

1. 打开 [https://jhkkv.github.io/----/](https://jhkkv.github.io/----/) 
2. 点击「设置」→ 填入你的 **DeepSeek API Key** 
   - 👉 [免费注册获取API Key](https://platform.deepseek.com/)（注册即送500万token）
3. 上传简历（支持 PDF / Word / TXT），AI自动解析背景
4. 从BOSS直聘复制岗位JD → 粘贴到输入框
5. 点击「生成招呼语」→ AI生成3个版本供你选择
6. 点击「复制」→ 切换回BOSS直聘 → 粘贴到聊天框 → 手动发送

### 方式二：本地使用

1. 下载 [`index.html`](index.html) 文件
2. 双击用浏览器打开
3. 同上步骤使用

### 方式三：安装为手机App

1. 在手机浏览器打开上述链接
2. 点击浏览器菜单 → **「添加到主屏幕」**
3. 下次直接从桌面图标打开，像App一样使用

## 🔒 隐私说明

- 你的**简历文件在浏览器本地解析**，不上传到任何服务器
- 发送到AI API的文本**已自动脱敏**（移除姓名、手机号、邮箱等个人信息）
- API Key 存储在浏览器本地，仅用于API调用
- 所有数据可通过「清除数据」一键删除

## 🛠️ 技术栈

- 纯 HTML + CSS + JavaScript，零依赖
- PWA (Progressive Web App)，支持离线缓存
- 暗色 Teal 主题，移动端优先设计
- 支持 PDF.js / Mammoth.js 本地文件解析
