# Pixel Runner — 像素跑酷游戏设计文档

## 概述

一款像素风横版跑酷游戏，类似超级马里奥的视觉风格，玩法为自动跑步 + 跳跃躲避。纯浏览器运行，单 HTML 文件实现，排行榜基于 localStorage。

## 技术方案

- **技术栈**：单个 HTML 文件（HTML + CSS + JavaScript）
- **渲染**：Canvas 2D API
- **存储**：localStorage（本地排行榜）
- **运行方式**：双击 HTML 文件即可在浏览器中打开，手机浏览器也可

## 游戏玩法

| 项目 | 描述 |
|------|------|
| 角色移动 | 自动向右跑，速度逐渐加快 |
| 操作方式 | PC：空格键跳跃 / 手机：点击屏幕跳跃 |
| 障碍物 | 随机出现的尖刺方块，碰到即游戏结束 |
| 金币 | 随机分布，吃到加分 |
| 计分 | 跑步距离分 + 金币收集分 = 总分 |
| 结束条件 | 撞到障碍物 |

## 视觉设计

| 元素 | 描述 |
|------|------|
| 角色 | 16×16 小红帽像素小人，2 帧跑步动画 |
| 地面 | 绿色草地 + 棕色泥土像素纹理 |
| 障碍物 | 深色方块 + 顶部尖刺 |
| 金币 | 黄色闪烁小方块 |
| 背景 | 蓝天 + 像素云朵漂移 |
| UI | 左上角像素字体实时分数 |
| 色板 | NES 经典 8-bit 配色 |

## 排行榜

| 项目 | 描述 |
|------|------|
| 存储 | localStorage |
| 记录 | 分数、日期、玩家名字 |
| 显示 | TOP 10，按分数降序 |
| 名字输入 | 游戏结束时输入 |
| 重置 | 底部"清空排行榜"按钮 |

## 文件结构

```
pixel-runner/

```

## 核心模块划分

1. **游戏常量与配置** — 速度、重力、跳跃力度、颜色等
2. **游戏状态管理** — 待机/运行中/结束，分数追踪
3. **角色** — 位置、速度、跳跃物理、动画帧
4. **障碍物生成** — 随机间隔生成，碰撞检测
5. **金币生成** — 随机位置生成，收集检测
6. **背景渲染** — 天空、云朵、地面像素纹理
7. **UI 渲染** — 实时分数、开始画面、结束画面
8. **排行榜系统** — localStorage 读写、排序、显示
9. **输入处理** — 键盘 + 触摸事件
10. **游戏循环** — requestAnimationFrame 驱动

## 代码规模预估
