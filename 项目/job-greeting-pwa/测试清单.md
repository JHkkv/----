# BOSS直聘AI招呼语生成器 -- 手动测试清单

> **测试目标**：确保单文件 HTML PWA 在桌面端和移动端均正常工作，覆盖正常流程、边界场景、离线模式和PWA特性。
> **测试方法**：手动操作 + 浏览器开发者工具验证。

## 一、环境准备

- [ ] Chrome 桌面版（最新稳定版）
- [ ] Chrome Android 或 iOS Safari（移动端真机）
- [ ] 浏览器开发者工具已打开（Console 和 Network 面板）

## 三、边界场景与异常处理

### 3.1 输入验证

- [ ] **空 JD 提交**：不粘贴 JD 直接点生成 → 显示"请先粘贴岗位JD"错误提示，JD 输入框高亮红色
- [ ] **JD 过短**（< 20 字）：输入"招运营" → 显示"JD文本太短"错误提示
- [ ] **JD 字数刚好 20 字**：粘贴 20 字 JD → 验证通过，无错误提示
- [ ] **JD 清空后输入**：先收到错误提示，再输入足够长度 → 错误提示自动消失
- [ ] **无技能标签**：不添加任何技能标签 → 生成（当前设计仍允许生成）→ 正常处理
- [ ] **无关键经历**：不填写关键经历 → 生成正常进行，Prompt 中标记为"（未填写关键经历）"

### 3.2 API 相关错误

- [ ] **未配置 API Key**：清空 API Key → 点击生成 → 提示"请先在设置中配置API Key"
- [ ] **无效 API Key**（401）：使用错误 Key → 提示"API Key无效（401）"
- [ ] **API 频率限制**（429）：短时间内多次生成触发限制 → 提示"请稍后重试"
- [ ] **API 服务器异常**（500+）：模拟或等待服务器异常 → 提示"API服务异常"
- [ ] **API 返回空内容**：AI 返回空 choices → 提示"AI返回内容为空"
- [ ] **API 返回格式异常**：AI 返回非标准 JSON → 触发解析流程（尝试3种解析方式）
- [ ] **API 超时**（> 30 秒）：网络极慢或 API 挂起 → 超时错误提示
- [ ] **API Endpoint 错误**：填错 API 地址 → 显示网络错误提示

### 3.3 网络异常

- [ ] **断网状态生成**：断开网络 → 点击生成 → 提示"当前处于离线模式"
- [ ] **生成中断网**：开始生成后立即断网 → 显示网络错误提示
- [ ] **恢复网络后重试**：断网 → 重新连接 → 点击生成 → 正常生成
- [ ] **网络慢**：使用 Chrome DevTools 限制网速（Slow 3G）→ 按钮显示 loading 直到 API 返回

### 3.4 数据持久化异常

- [ ] **localStorage 满**：模拟填充 localstorage 到上限 → 保存操作不崩溃，静默降级
- [ ] **localStorage 禁用**：浏览器隐私模式下 localstorage 不可用 → 页面仍可加载，功能降级但不崩溃
- [ ] **损坏数据恢复**：手动写入格式错误的 localStorage 数据 → 页面加载不崩溃，使用默认值
- [ ] **数据迁移**：修改 SCHEMA_VERSION → 旧数据正确迁移（如果实现了迁移逻辑）
