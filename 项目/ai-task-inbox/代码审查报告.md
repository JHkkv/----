#### 3. `classifyWithAI` 缺少 fetch 超时控制

**位置**: 第 1211-1255 行

**问题**: Fetch 请求没有 `AbortController` 超时机制。PRD 明确要求 "API 超时（>10秒）→ 中止请求，提示 AI 响应超时"。当前实现如果 API 服务端挂起不响应，加载状态（spinner + 按钮 disabled）会永久卡死，用户无法恢复。

**修复建议**:
```js
var controller = new AbortController();
var timeout = setTimeout(function () { controller.abort(); }, 15000);
fetch(endpoint, { /*...*/, signal: controller.signal })
  .finally(function () { clearTimeout(timeout); });
```

#### 6. `escapeHtml` 用于 HTML 属性值但未转义双引号

**位置**: 第 1049-1053 行 (escapeHtml 定义) + 第 1306 行 (使用点)

```js
function escapeHtml(str) {
  var div = document.createElement('div');
  div.appendChild(document.createTextNode(str));
  return div.innerHTML;
}
// ...
html += '... data-cat="' + escapeHtml(cat) + '">';
```

**问题**: `createTextNode` + `innerHTML` 转义 `<`、`>`、`&`，但**不转义双引号**。当分类名包含 `"` 时（例如用户创建名为 `work"test` 的分类），会破坏 `data-cat="..."` 属性语法，导致 HTML 结构错误。这是自我 XSS（本应用仅本地使用，实际风险极低），但作为编码模式，若被复制到多用户场景会引入安全漏洞。

**修复建议**: 
- 方案 A: 使用属性安全的转义，将 `"` 替换为 `&quot;`
- 方案 B（推荐）: 对于 `data-cat` 属性使用 `element.dataset.cat = cat` 而非字符串拼接，彻底避免注入

**位置**: 第 1801-1803 行 (`doClassify` 函数入口)


### LOW（2 项）

#### 9. `uuid()` 使用非加密级随机数

**位置**: 第 1055-1057 行

```js
function uuid() {
  return Date.now().toString(36) + Math.random().toString(36).slice(2, 8);
}
```

**问题**: 使用 `Math.random()` 而非 `crypto.randomUUID()` 生成 ID。对于单用户本地应用碰撞概率可忽略，但 PRD 明确建议使用 `crypto.randomUUID()`。

**修复建议**: 
```js
function uuid() {
  return crypto.randomUUID ? crypto.randomUUID() : Date.now().toString(36) + Math.random().toString(36).slice(2, 8);
}
```

#### 10. Esc 键关闭多个面板无优先级控制

**位置**: 第 1990-1998 行

```js
document.addEventListener('keydown', function (e) {
  if (e.key === 'Escape') {
    closeSettings();
    closeEditModal();
    hideTabEditRow();
    hideApikeyPrompt();
    dom.tabContextMenu.classList.remove('visible');
  }
});
```

**问题**: 按 Esc 会同时尝试关闭 5 个面板，而不是按层级"先关最上层"。虽然每个 `close` 操作都是幂等的（对已隐藏的元素无副作用），但这种"全关"策略在某些场景下不理想 -- 比如用户在设置面板中打开了 API Key 提示窗，按 Esc 本应只关提示窗，但实际会把设置面板也关了。

**修复建议**: 按优先级只关闭最上层的面板（编辑模态框 > API Key 提示 > 设置面板 > 上下文菜单 > 编辑行），而非全量关闭。
