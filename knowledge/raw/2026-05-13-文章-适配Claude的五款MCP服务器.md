# 适配 Claude 的五款实用 MCP 服务器

## 内容

### Perplexity MCP
可让 Claude 在不脱离工作流程的前提下，实时搜索整个互联网。

### Playwright MCP
赋予 Claude 完整浏览器控制权，能快速完成填写表单、上传视频、测试网页应用等操作。

### Fire Crawl MCP
支持 Claude 抓取任意网站内容，并直接加载至对话界面。

### Glyph MCP
可连接 Claude 与数百个图像、视频生成模型，还能自动针对每个模型优化提示词。

### Chrome MCP
已内置在 Claude Code 中，能让 Claude 查看并交互谷歌浏览器内的已打开内容。
