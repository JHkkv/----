# Coze 从零基础到商业落地学习路径

## 一、入门基础搭建
- 认识 Coze 界面
- 区分智能体（互动聊天）与工作流（自动化流程）
- 理解变量作用
- 实操搭建首个智能体

## 二、数据结构攻坚
- 学习数据类型：字符串、数字、数组、对象
- 实操项目：产品营销器、公众号快讯生成、天气 API 解析

## 三、逻辑记忆升级
- 掌握判断（电商客服路由）
- 循环（长篇小说生成）
- 数据库增删改查
- 让工作流具备判断、循环、持久记忆能力

## 四、大模型深度应用
- 理解 token 与上下文机制
- 调优大模型参数（温度、top p 等）
- 掌握工具调用、知识库搭建、三类提示词工程
- 实操多个自动化项目

## 五、商业落地突破
- HTTP 调用对接外部模型
- 图像生成（电商主图等）
- 视频生成（爆款复刻等）
- 语音播客制作
- 实现多模态自动化

## 六、两大专题拓展
- Coze 与飞书多维表格联动，搭建企业级自动化系统
- AI 辅助编程，指挥 AI 生成调试代码，无需懂编程即可搭建复杂工作流

## 学习建议
- 每天投入 1-2 小时
- 约 2 个月可完成学习
- 方法：先实操再理解、学后复现、关联实际场景
- 避免碎片化学习
