# MCP 服务器

## 定义
MCP（Model Context Protocol）是一种让 AI 助手连接外部工具的标准协议。就像手机上的 USB 接口——一个统一的标准，插什么外设都能用。

## 大白话解释
Claude 本身只能读文字、写文字。但通过 MCP 服务器，它可以：
- 联网搜索（本来不能上网）
- 操控浏览器（本来只能看不能点）
- 抓取网站（本来只能读你发给它的链接）
- 生成图片和视频

**MCP 就是 Claude 的"外接装备接口"。**

## 来源
> MCP 服务器可让 Claude 在不脱离工作流程的前提下，获得搜索、浏览、抓取、生成等扩展能力。

## 五个典型 MCP

| MCP 名称 | 赋予 Claude 的能力 | 类比 |
|----------|-------------------|------|
| Perplexity | 实时搜索整个互联网 | 给 Claude 装了个搜索引擎 |
| Playwright | 完整浏览器控制权（填表、上传、测试） | 给 Claude 装了一只手，能操作网页 |
| Fire Crawl | 抓取任意网站内容到对话界面 | 给 Claude 装了个搬运工 |
| Glyph | 连接数百个图像视频生成模型 | 给 Claude 装了个画师+摄影师 |
| Chrome | 直接查看和交互已打开的浏览器 | 给 Claude 装了一双眼睛，能看到你的浏览器 |
