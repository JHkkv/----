# 双向链接

## 定义
Wiki 概念页面之间互相引用的机制，使知识形成网络而非孤立条目，精准定位知识。

## 来源
> 大模型会自动提取核心概念、建立双向链接。

## 实现方式
- 新增概念时，列出指向已有概念的 `[[链接]]`
- 同时更新已有概念页面，添加回链
- 使用关联类型标签：`#延伸` `#对比` `#前提` `#应用`
- 使用 Obsidian 兼容的 `[[wiki/概念名]]` 格式

## 关联类型

| 标签 | 含义 |
|------|------|
| #延伸 | 概念 A 是概念 B 的延伸/具体化 |
| #对比 | 概念 A 与概念 B 形成对比/对立 |
| #前提 | 概念 A 是理解概念 B 的前提 |
| #应用 | 概念 A 在概念 B 中被应用 |
