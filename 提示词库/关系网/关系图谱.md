# 🔗 提示词关系图谱

## 概述

本页是提示词关系网络的入口。在 Obsidian 的 **Graph View** 中，可以直观看到：

- 🟢 **绿色节点**：`新提示词/` 中的活跃提示词
- 🟡 **橙色节点**：`旧提示词/` 中的归档提示词
- 🔵 **蓝色节点**：关系网说明页面

节点之间的连线表示提示词的关联关系。

## 关联类型

### 相似关联
功能相近、可互相替代的提示词。
```
prompt-A ---相似--- prompt-B
```

### 互补关联
配合使用效果更好的提示词组合。
```
prompt-C ---互补--- prompt-D
```

### 升级关联
从旧版本优化而来的新版本。
```
prompt-E ---升级---> prompt-F
```

### 组合关联
可以合并为更强大提示词的条目。
```
prompt-G ---组合--- prompt-H
              ↓
         prompt-GH（合并版）
```

## 使用建议

1. 打开 Obsidian Graph View（快捷键 `Ctrl+G`）
2. 使用颜色分组区分新旧提示词
3. 点击节点查看提示词详情
4. 沿着连线发现相关提示词组合

## 维护规则

- 每次新增提示词时，检查是否有关联的现有提示词
- 优化周期中自动检测潜在关联
- 合并相似度过高的提示词
