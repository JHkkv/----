# 多Agent并行工作流编排

## 提示词内容

> 我需要完成 [复杂任务]。请设计一个多 Agent 并行工作流：
>
> 1. 将任务分解为独立子任务（每个子任务可由单独 Agent 执行）
> 2. 对每个子任务，写出对应的 Agent 提示词
> 3. 指定子任务之间的依赖关系（哪些可以并行，哪些需要串行）
> 4. 指定聚合策略：如何合并多个 Agent 的输出
> 5. 指定验证策略：如何检查最终结果的正确性
>
> 使用以下 Agent 类型：[列出可用 Agent 类型]。每个 Agent 提示词需包含：角色、任务、输出格式、质量要求。
