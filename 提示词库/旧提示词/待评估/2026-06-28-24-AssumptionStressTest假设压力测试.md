# Assumption Stress Test - 假设压力测试

## 提示词内容

> List every assumption this argument relies on. For each assumption: 1. Rate how fragile it is (High/Medium/Low); 2. Explain what evidence would be needed to validate it; 3. Describe what changes if this assumption turns out to be wrong. Then tell me which 2-3 assumptions, if proven false, would cause the entire argument to collapse.

## 使用场景

- 商业论证的可靠性检验
- 学术论文论点审查
- 项目计划风险评估
- 辩论和论证准备

## 适用模型

- ChatGPT / Claude / Gemini 等通用 LLM
