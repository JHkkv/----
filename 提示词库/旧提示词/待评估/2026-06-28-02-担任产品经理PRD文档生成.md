# 担任产品经理 - PRD 文档生成

## 提示词内容

> 请您作为产品经理回复我。我将会提供一个主题，您将帮助我编写一份包括以下章节标题的PRD文档：主题、简介、问题陈述、目标与目的、用户故事、技术要求、收益、KPI指标、开发风险以及结论。在我要求具体主题之前，请不要写任何内容。

## 使用场景

- 新产品功能规划
- 快速产出标准化 PRD 文档
- 产品需求梳理和结构化表达
- 团队对齐需求理解的参考文档

## 适用模型

- ChatGPT / Claude / Gemini 等通用 LLM
