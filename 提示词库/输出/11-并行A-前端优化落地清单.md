## 一、Hotfix Sprint（第 1 周，7 小时）

### 任务 1：移动端时间选择器触摸事件隔离

**文件**：`DatePickerPopover.tsx`
**问题**：移动端 Popover 内时间滚轮与外层滚动冲突
**改动**：
```tsx
// PopoverContent 增加触摸事件阻断
<PopoverContent
  onTouchMove={(e) => e.stopPropagation()}
  onWheel={(e) => e.stopPropagation()}
>
```
**验收**：移动端可流畅滚动选择时间，页面不跟随跳动
**工时**：2h
**依赖**：无

### 任务 3：排行榜中位数对比参考线

**文件**：`RankBarChart.tsx`
**问题**：缺少全用户中位数参考线，用户无法判断相对水平
**改动**：
```tsx
import { ReferenceLine } from 'recharts'

<BarChart data={rankData}>
  <XAxis dataKey="date" />
  <YAxis />
  <Bar dataKey="hours" fill="var(--accent)" radius={[4, 4, 0, 0]} />
  <ReferenceLine
    y={medianHours}
    stroke="var(--color-muted)"
    strokeDasharray="4 4"
    label={{
      value: `中位数 ${medianHours}h`,
      position: 'right',
      fill: 'var(--color-muted)',
      fontSize: 12,
    }}
  />
</BarChart>
```
**验收**：柱状图中可见虚线中位数线，标注文字清晰可读
**工时**：2h
**依赖**：后端需提供 `/api/leaderboard/median` 接口
