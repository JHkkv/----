# 江述

## 一句话概述

32岁水利工程师穿越秦朝，从赵国窄巷救下少年嬴政开始，走到大秦权力棋盘的最核心，最终成为帝国崩塌时的"看客"。

## 穿越前

| 维度 | 内容 |
|------|------|
| 年龄 | 32岁 |
| 学历 | 大学历史系毕业，未走学术道路 |
| 职业 | 水利工程师，从业七年 |
| 军历 | 两年兵役（后勤与军事工程方向，非战斗） |
| 爱好 | 秦史/兵法，藏有《史记》《战国策》《孙子兵法》批注本 |

### 穿越方式

> 某夜读一本从旧书摊淘来的残破古籍（无题，内容为从未见过的秦代记载），读到某页时字迹发光。睁眼时已身处赵国巷陌——少年嬴政正遭人追杀。

## 穿越后身份

1. **赵地游学士子**（初入秦）
2. **客卿**（嬴政即位后）
3. **郎中将**（统一前夕）
4. **太子扶苏之师**（统一治世）
5. **游历者**（嬴政死后，退出朝堂，旁观秦末大乱）

## 性格

| 维度 | 描述 |
|------|------|
| 处事风格 | 务实冷静。七年工程师经历→先评估可行性，再动手 |
| 人情世故 | 圆融不圆滑。能跟粗人喝大酒，也能跟上位者汇报 |
| 历史态度 | 非教条派。承认历史惯性（统一为大势），相信局部可改变 |
| 底线 | 不行无差别杀戮。有些秦军手段他不会赞同 |
| 冷幽默 | 心里吐槽（秦法太严→"这也太卷了"），嘴上不说 |
| 情感软肋 | 与嬴政的羁绊最深——少年之交，但嬴政终究是秦始皇 |

## 知识优势

- 水利工程（专业）→ 郑国渠改良
- 基础农业技术 → 农具改良、轮作
- 军事后勤（部队经验）→ 秦军补给线优化
- 粗通冶炼（工程师）→ 精钢的前期铺垫
- 兵法爱好者 → 能与王翦等名将对话

## 知识盲区

- 不会炼丹/修真的具体法门（靠系统解锁）
- 不懂秦代官场礼仪细节（需时间适应）
- 墨家/阴阳家专业知识不足
- 化学/医学仅常识水平
