# 李斯

## 一句话概述

丞相，朝堂合作者/竞争伙伴。天资极高但底线模糊。与江述从合作走向裂痕。

## 关键事件

| 事件 | 说明 |
|------|------|
| 入秦为舍人 | 从楚国来秦，与江述初次见面，彼此认可但也互相审视 |
| 韩非之死 | 抢先害死韩非，与江述之间第一道裂痕 |
| 《谏逐客书》 | 阻止嬴政驱逐六国士人，江述公开支持 |
| 沙丘之变 | 与赵高合谋矫诏，赐死扶苏 |

## 与江述的关系

- 合作者/竞争伙伴
- 江述认可他的天资，但对其底线模糊感到不安
- 韩非之死是二人关系转折点
