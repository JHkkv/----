# 镇秦璧

## 一句话概述

系统奖励的最后一件法宝。秦式玉璧，正面山河纹，背面小篆"秦"字。嬴政死后魂魄寄存其中。

| 维度 | 内容 |
|------|------|
| **器形** | 秦式玉璧，正面山河纹，背面小篆"秦"字 |
| **来源** | 系统奖励的最后一件法宝 |
| **功能** | 嬴政以器灵形态寄存，能感知外界、与江述对话，无实体无战力 |
| **关系** | 平等朋友。嬴政不再是皇帝，江述不再是臣子 |
| **风险** | 璧裂则魂散。嬴政自己选择何时消散 |
