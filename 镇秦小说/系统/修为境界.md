# 修为境界

## 一句话概述

参考金庸武侠体系 + 轻玄幻。无夸张飞天遁地，更多是延年益寿 + 提升武力值。基础武学招式存在。

## 七境修炼

| 境界 | 对应章数 | 效果 |
|------|----------|------|
| **筑基** | 1-25章 | 体能超常人约2倍。拳可碎石 |
| **通脉** | 26-60章 | 内力初成。单挑五六人 |
| **凝丹** | 61-120章 | 内力精纯。可敌精锐武将 |
| **化神** | 121-220章 | 内力外放。单挑数十人 |
| **归真** | 221-310章 | 气机圆融。可敌一品武将 |
| **天人** | 311-380章 | 天人感应。一人可退一军 |
| **陆地神仙** | 381-400章 | 凡躯之极。历史上仅少数人达到 |

## 突破机制

- 每满100分山河策面板升一小阶
- 500分升中阶
- 1000分满值对应大境界突破
- 每个大境界跨越需对应足够的历史事件积累，不跳跃
