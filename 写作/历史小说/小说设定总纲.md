## 一、基础信息

| 项目 | 内容 |
|------|------|
| 书名 | **镇秦** |
| 时代 | 秦朝（前247→前206） |

## 二、主角：江述

### 穿越前

| 项目 | 内容 |
|------|------|
| 年龄 | 32岁 |
| 学历 | 大学历史系毕业，未走学术道路 |
| 职业 | 水利工程师，从业七年 |
| 军历 | 服过两年兵役（后勤与军事工程方向，非战斗专精） |
| 爱好 | 秦史/兵法。家中藏有《史记》《战国策》《孙子兵法》的批注本 |

### 穿越方式

某夜读一本从旧书摊淘来的残破古籍（无题，内容为从未见过的秦代记载），读到某页时字迹发光。睁眼时已身处赵国巷陌——少年嬴政正遭人追杀。

### 穿越后身份

- 救下嬴政后，以"赵地游学士子"身份入秦
- 嬴政即位后，成为其身边最年轻的客卿之一
- 统一前夕逐步升迁至郎中将（皇帝近臣）
- 统一治世期间被任命为太子扶苏之师
- 嬴政死后退出朝堂，以游历者身份旁观秦末大乱

### 性格

| 维度 | 描述 |
|------|------|
| 处事风格 | 务实冷静。七年工程师经历→先评估可行性，再动手 |
| 人情世故 | 圆融不圆滑。能跟粗人喝大酒，也能跟上位者汇报。穿越后快速适应秦朝环境 |
| 历史态度 | 非教条派。承认历史惯性（统一为大势），相信局部可改变 |
| 底线 | 不行无差别杀戮。有些秦军手段他不会赞同 |
| 冷幽默 | 心里会吐槽（秦法太严→"这也太卷了"），嘴上不说 |
| 情感软肋 | 与嬴政的羁绊最深——少年之交，但嬴政终究是秦始皇 |

### 知识优势（系统外自带）

- 水利工程（专业）→ 郑国渠改良
- 基础农业技术 → 农具改良、轮作
- 军事后勤（部队经验）→ 秦军补给线优化
- 粗通冶炼（工程师）→ 系统奖励精钢前可做前期铺垫
- 兵法爱好者 → 能与王翦等名将对话

### 知识盲区（避免全能）

- 不会炼丹/修真的具体法门（靠系统解锁）
- 不懂秦代官场礼仪细节（需时间适应）
- 墨家/阴阳家专业知识不足（需向方士群体学习）
- 化学/医学仅常识水平

## 三、系统：山河策

### 机制

A+B+C 三者融合：

**量化面板（B）**：五项基础指标，每项 0-1000 分。

| 指标 | 影响领域 | 提升方式 |
|------|----------|----------|
| 民生 | 农业、赋税、人口 | 兴修水利、减赋、赈灾 |
| 军力 | 兵员素质、装备、士气 | 练兵、改良兵器、落实军功爵 |
| 政治 | 吏治、法令、朝堂稳定 | 举贤、肃贪、平衡宗室 |
| 文教 | 识字率、典籍、人才 | 推广简牍、改良书写工具 |
| 国运 | 综合国力，嬴政信任 | 随前四项联动增长，不可直接提升 |

每满100分升一小阶修为，500分升中阶，1000分满值对应大境界突破。

**任务触发（C）**：
- 天时之命（主线）：不可拒绝，与历史走向挂钩。完成得大奖励，失败扣修为
- 人道之任（支线）：民生改良类，可选择接受
- 隐机之缘（隐藏）：与人物命运相关，触发条件隐藏

**里程碑奖励（A）**：

| 节点 | 奖励 |
|------|------|
| 救嬴政/系统激活 | 山河策启动 |
| 嬴政亲政（前238） | 修为第一重大境界突破 + 山河简（预判天气农时） |
| 灭韩 | 修为突破 + 造纸术全套工艺 |
| 灭赵 | 修为突破 + 精钢冶炼法 |
| 统一天下（前221） | 大境界突破 + 帝道之玺（沟通国运与修为的核心法器） |
| 扶苏拜师 | 开启收徒系统 |
| 始皇崩殂 | 系统静默期，修为封存 |
| （隐藏）扶苏未死 | 系统重启，修为解封 |

### 核心限制

- 只奖励知识方向和修为，不能直接改变他人意志或凭空变出军队
- 每次奖励消耗精神力，过度使用会昏迷
- 对历史主线有保护机制 — 不能阻止统一大势
- **出现频率**：每2-3章出现一次，不在每章刷存在感。危机时刻系统的重要性更高
- **前期法宝**：给几个护身型（非攻击型），江述不主动展示

## 三-B、武力值与法宝

### 明暗双线

| 层面 | 表现 | 说明 |
|------|------|------|
| **明面** | 习过武但不精通，勉强自保 | 对外展示的形象——"会点拳脚但不算高手" |
| **私下** | 随修为提升逐步增强，后期无人能敌 | 只在生死关头展露真实实力 |

### 原则
- **藏拙**：从不主动展示真实武力。被挑衅时能忍则忍，忍无可忍才出手且只展示刚好够用的程度
- **护身法宝**：系统前期奖励几件护身型法宝（非攻击型），不引人注目。如：护心镜、替身符、预警玉
- **提升节奏**：与修为同步，不跳跃。每次大境界突破武力值有明显提升

## 四、法器：镇秦璧

- **器形**：秦式玉璧，正面山河纹，背面小篆"秦"字
- **来源**：系统奖励的最后一件法宝。嬴政临终时江述以此留存其魂魄
- **功能**：嬴政以器灵形态寄存，能感知外界、与江述对话，无实体无战力
- **关系**：平等朋友。嬴政不再是皇帝，江述不再是臣子
- **风险**：璧裂则魂散。嬴政自己选择何时消散
- **末章意义**：嬴政以器灵之眼目睹帝国崩塌 → "看客"的第二层含义。第三卷核心设定。

## 五、修为境界体系

参考金庸武侠体系 + 轻玄幻。无夸张飞天遁地，更多是延年益寿 + 提升武力值。基础武学招式存在。

| 境界 | 对应章数 | 效果 |
|------|----------|------|
| 筑基 | 1-25章 | 体能超常人约2倍。拳可碎石 |
| 通脉 | 26-60章 | 内力初成。单挑五六人 |
| 凝丹 | 61-120章 | 内力精纯。可敌精锐武将 |
| 化神 | 121-220章 | 内力外放。单挑数十人 |
| 归真 | 221-310章 | 气机圆融。可敌一品武将 |
| 天人 | 311-380章 | 天人感应。一人可退一军 |
| 陆地神仙 | 381-400章 | 凡躯之极。历史上仅少数人达到 |

## 六、与嬴政的关系（三阶段）

| 阶段 | 卷/章数 | 关系 |
|------|------|------|
| 第一卷·入秦 | 1-100 | 绝对信任。从赵国窄巷救下的少年到统一天下的帝王——每一次献策带来国力增长，嬴政视为"天赐之才" |
| 第二卷·治世 | 101-300 | 信任+猜忌并存。江述无明显衰老引发嬴政疑虑；劝阻政策时会引起不满但不会被杀。二人间隙逐步加深，但江述始终视他为当年马厩里的少年 |
| 第三卷·余烬 | 301-400 | 嬴政崩殂→魂魄入璧。失去权力后回归平等，以器灵之眼目睹帝国崩塌。最终二人相视一笑——恩怨尽了 |

**安全线**：嬴政一生未杀开国功臣 — 江述即使与他冲突也不会被处死。

## 九、写作风格（番茄平台适配）

### 核心调整（第21章起执行）

| 维度 | 旧问题 | 新方向 |
|------|--------|--------|
| 主角风格 | 扁平，"工具人" | 内心疯狂吐槽、嘴上滴水不漏的现代工程师 |
| 趣味性 | 过于严肃 | 加入穿越者遭遇古代日常的反差趣味 |
| 节奏 | 平淡，章尾自然收束 | 每章结尾必须有钩子 |
| 爽点密度 | 偏低 | 每3-5章一大爽，每章一微爽 |

### 具体手法
- **内心吐槽**：现代视角看古代的反差。黍米饭→"嚼砖感"；秦法严苛→"这也太卷了"
- **藏拙**：武力不主动展示，但让读者知道"他其实很强"
- **钩子**：章尾卡在未说完的话、敲门声、面板预警——让读者想翻下一章
- **人设＞剧情**：读者先记住"这人有趣"，才关心"他做什么"
- **权谋精细**：对话每句有潜台词，提前多章埋线索

## 七、与扶苏的关系

- 江述为扶苏之师（C29正式开始授课）
- 在扶苏身上看到秦朝未来的希望 — 仁厚、宽和
- 扶苏谏言被贬 → 江述作为老师无力干预 → 全书痛感点之一（第二卷）
- 扶苏之死（前210）：江述得知时已晚 → 最痛的情感高点（第二卷末）
