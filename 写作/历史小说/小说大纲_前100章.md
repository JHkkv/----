# 《镇秦》前100章大纲 — 统一前夕

## 第一弧：天降（第1-5章）

| 章 | 核心内容 |
|----|----------|
| 1 | **穿越**。江述读残破古籍时字迹发光，睁眼已在赵国巷陌。少年嬴政正被几名追杀者围堵。江述用现代格斗基础+环境利用击退追杀者，自己也受伤。 |
| 2 | **初识**。江述与嬴政躲在一处废弃马厩中。嬴政（约12-13岁）虽年幼但眼神不似孩童。两人交谈——江述得知自己穿越到了秦庄襄王时期。嬴政问他"你是何人"——江述答"路过的人"。 |
| 3 | **山河策激活**。嬴政被江述所救后，江述脑中响起系统提示音。山河策首次激活——"救秦王子，天时之命完成。"系统面板展开：民生/军力/政治/文教/国运五项指标。首奖励：体魄强化（筑基）。 |
| 4 | **赵姬**。嬴政之母赵姬派人寻到两人。江述以"赵地游学士子"身份被带回。庄襄王子楚接见——江述的谈吐和学识让子楚另眼相看。 |
| 5 | **留下**。子楚问江述是否愿留在秦国。江述知道即将发生什么（子楚三年后卒，嬴政即位），选择留下。获赐一匹瘦马、一处偏院、一册秦律入门。 |

## 第二弧：潜伏（第6-20章）

| 章 | 核心内容 |
|----|----------|
| 6-8 | **适应秦朝**。江述学习秦代官场礼仪+文字。与少年嬴政互动——教他一些基础的地理/历史知识。嬴政对这个"什么都懂一点"的年轻士子产生好奇。 |
| 9-10 | **初识吕不韦**。吕不韦考验这个新来的"赵地士子"。江述在相国面前低调行事，但在一次水利问题上给出精准建议→吕不韦对他产生戒备但也认可其才能。 |
| 11-13 | **子楚病重**。庄襄王子楚健康状况恶化。嬴政频繁来找江述。江述知道历史——子楚将在前247年去世。他无法改变这一点，只能在旁陪伴。 |
| 14-15 | **即位**。前247年，秦庄襄王卒。嬴政即位，年仅13岁。吕不韦以相国身份摄政。江述选择不急于参与朝政——他需要更多时间了解这个时代。 |
| 16-18 | **郑国渠**。韩国派水工郑国入秦，游说修渠（"疲秦"之计）。江述以专业知识判断：此渠若修成将成为秦东征的粮仓。他支持郑国修渠，同时暗中向嬴政说明这是双刃剑——消耗秦力但长远受益。嬴政选择信任江述的判断。 |
| 19-20 | **成长**。嬴政逐渐长大，对吕不韦的专权感到不满。江述成为他私下交谈的唯一对象。江述向嬴政灌输一个观念——你需要自己的团队。 |

## 第三弧：青云（第21-40章）

| 章 | 核心内容 |
|----|----------|
| 21-23 | **江述献策**。江述首次向嬴政呈递正式策论——关于关中水利改良和军粮储备。嬴政阅后亲自登门与他彻夜长谈。系统面板政治指标首次破100→修为累积提升，但尚未破境。 |
| 24-26 | **嫪毐登场**。赵姬宠幸嫪毐，嫪毐开始在宫中培植势力。江述察觉到危险，但身为外人无法直接干预宫廷内事。他选择向嬴政暗示"有些人需要早做防备"。嬴政沉默不语。 |
| 27-30 | **亲政前夕**。前238年，嬴政即将亲政。江述协助嬴政制定亲政后的首批人事安排——提拔一批年轻士子进入朝堂基层。系统发布天时之命：助嬴政平定嫪毐之乱。 |
| 31-33 | **嫪毐之乱**。嫪毐发动叛乱。江述参与平叛——他负责协调咸阳城防的后勤，确保叛军无法突破关键关隘。平叛成功→系统大奖励：修为突破到**通脉**+**山河简**（可预判天气/农时）。 |
| 34-36 | **罢吕不韦**。嬴政罢免吕不韦相国之位。江述在朝堂上的地位从"赵地客卿"变为"秦王近臣"。吕不韦罢相后曾私下约见江述——交谈中吕不韦流露出复杂情绪。 |
| 37-40 | **组建班底**。嬴政开始组建自己的核心团队。江述推荐了几名有才能的基层官吏。李斯也在此时从楚国来到秦国成为舍人——江述与李斯初次见面，彼此认可但也互相审视。 |

## 第四弧：蓄势（第41-60章）

| 章 | 核心内容 |
|----|----------|
| 41-43 | **统一战争前的准备**。江述向嬴政呈递《备战策》——关于军粮储备、驰道预修、兵器标准化。嬴政全部采纳→系统政治+军力面板提升。江述首次正面参与军事会议，与王翦认识。 |
| 44-46 | **韩非入秦**。韩非作为韩国使节入秦。嬴政读其文章后大赞。江述想救韩非（他知道韩非将被李斯害死），但李斯动作更快→韩非入狱。江述试图劝嬴政，嬴政犹豫→李斯抢先→韩非死于狱中。这是江述与李斯之间第一道裂痕。 |
| 47-50 | **农业改革**。郑国渠建成。江述利用自己的水利专业知识改良关中灌溉系统→民生面板大幅提升。同时提供改良农具图纸→生产效率提升。修为突破到凝丹。 |
| 51-54 | **伐赵准备**。王翦提出伐赵策略。江述在后勤和情报上提供支持——他利用现代军事知识改良了补给线的组织方式。秦军首次感受到江述在军事后勤上的价值。 |
| 55-57 | **成蟜叛变**。嬴政之弟成蟜伐赵时投降赵国。嬴政震怒。江述私下劝嬴政——成蟜之叛背后可能有更复杂的政治原因。这是江述第一次看到嬴政身上冷酷的一面。 |
| 58-60 | **李斯崛起**。李斯上《谏逐客书》阻止嬴政驱逐六国士人。江述对此公开支持→嬴政收回成命。两人在朝堂上短暂合作→但江述也注意到李斯的政治野心远超自己。 |

## 第五弧：东出（第61-85章）

| 章 | 核心内容 |
|----|----------|
| 61-63 | **灭韩**（前230）。内史腾率军灭韩。江述参与后勤+情报工作→韩亡。系统里程碑奖励：造纸术。江述将造纸术通过系统转化为秦代可实现的工艺→文教面板大幅提升。 |
| 64-66 | **伐赵·李牧**。王翦攻赵受阻于李牧。江述建议用离间计——他提供了具体的方案（利用赵国内部的派系矛盾）。王翦采纳→李牧被赵王迁杀害。江述后来为此事感到不安——李牧是忠臣，死于阴谋非战之罪。 |
| 67-69 | **灭赵**（前228）。邯郸破，赵亡。江述随军入邯郸。他看到被俘的赵王迁，想起史书上那句"赵王迁素无行"。系统奖励→精钢冶炼法。 |
| 70-72 | **荆轲刺秦**（前227）。燕太子丹派荆轲刺秦王。江述当时在咸阳宫中。荆轲图穷匕见→嬴政绕柱走→江述情急之下掷出手中简牍击中荆轲手腕（非改变历史——正史中荆轲本就被众人制服）。事后嬴政震怒→下令攻燕。江述亲身经历了这次刺杀——他看到嬴政在生死关头的恐惧与暴怒。 |
| 73-75 | **攻燕**。王翦攻燕。燕王喜杀太子丹向秦求和。江述随军记录——他看到了战争的真实面貌。不是史书上的几行字，是真实的死亡。 |
| 76-78 | **灭魏**（前225）。王贲引黄河水灌大梁，魏王假投降。江述反对水攻——"大梁城中不只是军队，还有数十万百姓。"但王贲执行军令。江述第一次感受到自己和秦军战争机器之间的分歧。 |
| 79-82 | **李信之败**。李信率20万人伐楚，大败而归。江述参与战后复盘——他指出李信轻敌冒进，同时建议请王翦复出。嬴政亲自骑马去王翦家道歉请其出山→王翦索要田宅→江述明白这是在消除嬴政的猜疑。 |
| 83-85 | **灭楚**（前223）。王翦率60万大军灭楚。项燕、昌平君死。这是六国中抵抗最激烈的一战。江述负责协调倾国之兵的后勤——这是他穿越以来最忙碌也最疲惫的一段时间。修为精进至凝丹中阶。 |

## 第六弧：归一（第86-100章）

| 章 | 核心内容 |
|----|----------|
| 86-88 | **灭燕·灭代**（前222）。王贲攻辽东俘燕王喜，破代城俘代王嘉。燕、赵彻底灭亡。六国只剩下齐国。江述开始思考——统一之后是什么？ |
| 89-91 | **灭齐·统一天下**（前221）。王贲攻齐，齐王建不战而降。六国尽灭。嬴政在咸阳殿上接受群臣朝贺。江述站在百官中，知道自己见证了中国历史上最重要的一天。系统大里程碑：修为突破至**凝丹大成**+**帝道之玺**（沟通国运与修为的核心法器）。 |
| 92-94 | **称帝**。嬴政自称始皇帝。李斯上书废分封立郡县。江述支持郡县制（历史正确）。同时他私下向嬴政建议：六国旧贵族是隐患，但不宜全杀——分化瓦解为上。嬴政部分采纳。 |
| 95-97 | **统一制度**。书同文、车同轨、统一度量衡和货币。江述参与制定具体的标准化方案→系统政治+文教面板突破。造纸术开始小规模推广。 |
| 98-100 | **收束·前100章结尾**。江述站在咸阳宫的高台上俯瞰这座正在变成帝国中心的城市。二十多年过去了——从他在赵国巷陌救下那个少年，到此刻六国归于一统。他改变了什么？他没能改变什么？那些死去的人（韩非、李牧、无数无名的士兵和百姓）在他脑中掠过。系统面板闪了一下：国运突破500。下一阶段的挑战——不是统一，是治理。他摸了摸腰间那枚山河简——预判天气和农时的法宝正微微发烫。他知道，更难的在后面。 |

## 前100章节奏分布

| 弧段 | 章数 | 历史时间 | 江述修为 |
|------|------|----------|----------|
| 天降 | 1-5 | 前247前 | 筑基（系统激活基础奖励） |
| 潜伏 | 6-20 | 前247-240 | 筑基稳固 |
| 青云 | 21-40 | 前240-235 | 筑基→通脉（亲政平叛后破境） |
| 蓄势 | 41-60 | 前235-230 | 通脉→凝丹（农业改革完成后破境） |
| 东出 | 61-85 | 前230-223 | 凝丹初阶→中阶（每灭一国小幅精进） |
| 归一 | 86-100 | 前223-221 | 凝丹中阶→凝丹大成 |

> 化神延至第101-120章统一治世初期突破。每个大境界跨越需对应足够的历史事件积累，不跳跃。

### 每章字数

每章2500-3500字。前100章总字数预估：25万-35万字。
